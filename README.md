Resume Screener & Job Match Scorer (AI-Based)





An AI-powered Resume Screener that analyzes resumes, extracts key skills, and matches them against job descriptions using Natural Language Processing and Semantic Similarity techniques.
Project Overview

This project helps automate the hiring process by intelligently screening resumes and scoring candidates based on how well they match a given job description. It reduces manual effort and improves accuracy in candidate shortlisting.
Features
1.Upload and parse resumes (PDF support)
2.Extract skills using NLP techniques (NER)
3.Match resumes with job descriptions using semantic similarity
4.Generate match score for each candidate
5.Fast API backend for real-time processing

Tech Stack
Python 🐍
FastAPI 
spaCy (NER)
Hugging Face Transformers 🤗
Sentence Transformers (Semantic Similarity)
📌 How It Works
User uploads resume
System extracts text and skills using NLP
Job description is processed similarly
AI model calculates similarity score
Final match score is generated
